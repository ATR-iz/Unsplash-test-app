# Тестовое приложение для сервиса Unsplash.

В приложений используется следующий стек
> Kotlin  
> Dagger2  
> Retrofit2  
> Picasso  
> Kotlin Coroutines  
> Paging3  

Для работы приложения необходимо указать API key в модуле future-unsplash-impl, класса UnsplashServiceImpl.
